Модуль А

Приложение для управления проектами, для организации работы коллективов территориально распределенных сотрудников.

Осуществление таск-менеджмента

  • Создавать задачи с привязкой к территориям и адресам
  • Представление задач в виде «Списка» или «Карточек задач»
  • Гибкая настройка алгоритма выполнения работ Исполнителями при постановке задачи
  • Автоматический сбор отчетов по результатам выполнения задач
  • Представление задач на Карте

Создание задач с привязкой к территориям и адресам

  • Гибкая настройка Маршрута выполнения задачи по адресам необходимой территории

Отображение задач и результатов работ на картах

  • Отображать на Карте целевые показатели задач, параметры территорий, промежуточные результаты выполнения задач.

Проведение опросов любых типов, создание анкеты с помощью Конструктора анкет

  • В МП ««Модуль А» доступен «Конструктор анкеты» опроса, с возможностью задания различных типов вопросов, произвольного алгоритма проведения опроса
  • Автоматическое формирование Отчетов по результатам опроса в режиме онлайн

Обучение и тестирование сотрудников внутри приложения

  • Выставлять в «Модуль А» обучающие видео-материалы, инфографику, PDF-инструкции
  • Таргетировать обучающие материалы под различные целевые аудитории сотрудников (учитывая их знания, навыки и опыт)
  • Конструировать и запускать в приложении опросы в виде Тестов, с любой логикой движения пользователя внутри теста
